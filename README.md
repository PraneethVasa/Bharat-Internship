## TASK 1 - REGISTRATION FORM

        registration-form/
        │
        ├── models/
        │ └── User.js
        │
        ├── public/
        │ ├── index.html
        │ ├── styles.css
        │ └── scripts.js
        │
        ├── routes/
        │ └── user.js
        │

#### II. Install Dependencies
         npm install express mongoose body-parser bcryptjs
